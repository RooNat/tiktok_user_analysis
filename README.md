# 基于Python的抖音电商用户分析

- **项目背景**：2024年，抖音部分用户特征数据 数据由客观来源和用户自主填写组成，例如收入为用户自主填写，购买商品类型为用户花费最多的商品类型，同理总花费为购买同类商品的总花费。本项目使用Python的Pandas工具对抖音电商用户特征进行探索分析，并提出相应的策略。
- **数据集**：[抖音电商用户特征](https://tianchi.aliyun.com/dataset/204698)
- **参考项目**：[基于MySQL和Tableau的抖音电商数据分析](https://www.heywhale.com/mw/project/68bbfb9de715157629d35ce0)
